Demo1: display的默认初始值
下面使用自定义的custom标签来看一下默认的初始display属性值是什么？
很明显，custom标签的display属性满足以下几个特征：
1）标签的边界由内容的宽度撑开
2）非独占一行
3）不支持设置宽高
4）代码换行被解析成空格
很明显，这是display:inline的表现特征。
四个特征演示 custom:宽高边界特征 custom:非独占一行 custom:不支持设置宽高 custom:换行 被解析成空格
span:宽高边界特征 span:非独占一行 span:不支持设置宽高 span:代码换行 被解析成空格
