# Learning Timelines Index

# Introduction

这里是学习时间轴，用于记录整个持续的学习过程，以时间为维度帮助整理学习的过程。其主要的作用包括：

1. 帮助复习和强化记忆学习过的知识。

    在很多时候，很多知识在总结完后，如果不经过反复的强化记忆，过一段时间，记忆将变得越来越浅，以致过了很长一段时间后学习的内容将全部遗忘。这是一种浪费。

    可以通过学习的时间轴，结合当下流行的一些记忆方法，将学习过的知识强化到记忆中。
